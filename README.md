# Student-Management-System
Student Management System

# Description :
CRUD usage for Student Management System Using Java Spring Boot, JDK Java 17, Thymelead and Bootsrap. 

# Dependency :
1. Spring Boot DevTools
2. Lombok
3. Spring Web
4. MySQL Driver
5. Spring Data JPA
